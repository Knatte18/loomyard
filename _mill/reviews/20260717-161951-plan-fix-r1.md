## Fixed

- **[BLOCKING] "Card 8 healthy-pair scenarios never wire the _lyx junction"** — verified:
  `lyxtest.CopyPaired` copies the host/weft repos and resolves a `Layout` but does **not** wire
  the `_lyx` junction (so `PairInSync` would return "junction missing"), and `WireJunctions`
  rejects a pre-existing real host `_lyx`. Card 8 now requires calling
  `warpengine.WireJunctions(fixture.Layout, slug)` on the pristine fixture **before** seeding, then
  seeding `status.json` through the wired junction at `LoomStatusFile()`; added
  `internal/warpengine/junction.go` and `internal/warpengine/checkout_test.go` to Card 8 `Context`
  as the wire-then-operate pattern.
- **[MAJOR] "Card 7 defensive `!found` branch returns a nil error"** — verified: `ReadJSONStrict`
  returns `(zero,false,nil)` on `IsNotExist`, so the TOCTOU `!found` branch had a nil `rerr`;
  returning `Report{}, nil` would violate `OK == (len(Failures)==0)` and the escalate contract.
  Card 7 now specifies returning a **synthesized non-nil error** (`fmt.Errorf("loomengine: seed
  vanished between stat and read: %s", ...)`) for that branch.
- **[NIT] "Card 1 read-only / creates no directory is imprecise"** — Card 1 godoc/test wording
  tightened to "no `MkdirAll`; a sidecar `.lock` is still taken, so not fully side-effect-free",
  and the missing-file test now uses a missing file with an **existing** `t.TempDir()` parent so
  the miss is a clean `IsNotExist`, not a lock-acquire failure.
- **[NIT] "Card 8 not-a-git-repo scenario needs process-global chdir"** — Card 8 now notes the
  `not-a-git-repo` and `subdirectory` scenarios `os.Chdir` (+restore via `defer`/`t.Cleanup`) and
  MUST NOT run under `t.Parallel()`; all other scenarios drive `checkResolved(l)` with an injected
  Layout and need no chdir.
